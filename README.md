# 💻 Web Design Practice Portal

Welcome to my frontend development playground! This project is a multi-page web ecosystem designed to practice **Pure HTML & CSS Layouts**, modern CSS grouping, interactive popups, and clean user experience (UX) flows.

## 🌟 Features Included

*   **Interactive Main Hub (`index.html`):** A beautiful landing page featuring a full-screen hero image overlay, modern navigation layout, and dynamic response actions.
*   **Integrated Login Portal Modal:** Features a built-in login dialog popup card matching the styling of the **State Board of Technical Education, Bihar (SBTE-EMS)** portal.
*   **Dedicated Account Recovery (`recovery.html`):** A separated, highly polished, and professional account validation page layout.
*   **Amazing Sites Gallery (`page3.html`):** A custom structured 3x2 matrix gallery layout displaying world-famous destinations with descriptions.
*   **Responsive Styling:** Clean modern typography, sleek element borders, and custom CSS hover transition animations.

---

## 📂 Project Structure

    ├── index.html          # Main landing hub with interactive login dialog modal 
    ├── recovery.html       # Standalone SBTE-EMS forgotten password recovery module
    └── page3.html          # Clean destination matrix gallery with grid-aligned cards

---

## 🛠️ Tech Stack Employed

*   **HTML5:** Structure, custom semantic layers, forms, and validation tags.
*   **CSS3:** Flexbox layouts, modal overlay manipulation, custom positioning elements, overlay masking, and transitions.
*   **JavaScript (Vanilla):** Dynamic window styling class injection (`.active`) for custom popup display handling and functional alerts.
